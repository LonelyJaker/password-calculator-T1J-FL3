# password-calculator-T1J-FL3
# 🔐 动态密码计算器
**工程模式专用动态密码生成系统**

## 📖 项目简介
一款高颜值科技风前端网页密码计算器，专为车载/设备工程模式打造，自动根据**SOP出厂日期 + 当前实时时间**实时计算两组动态6位密码，支持小时自动刷新、日历选日期、一键复制密码，离线即可使用，无需后端服务。

## ✨ 核心功能
- 双密码并行计算：工程模式密码 + 加密区固定SOP密码
- 实时北京时间自动获取，**每小时自动更新密码**
- 内置可视化日历选择SOP出厂日期
- 输入框限制8位日期格式，合法校验
- 一键复制生成密码，带复制成功提示
- 显示当前计算时间、SOP设定时间、系统运行状态
- 可视化倒计时：显示下次密码更新时间
- 全响应式布局，电脑/手机均可流畅使用
- 赛博科技炫酷UI、动态背景流光特效
- 纯前端HTML+CSS+JS，**零依赖、免部署、直接打开即用**

## 🧮 密码计算规则
1. **工程模式动态密码**
`(自定义SOP日期 × 当前月日时 - 当前小时) 取最后6位数字`

2. **加密区固定密码**
`(固定出厂SOP：20250830 × 当前月日时 - 当前小时) 取最后6位数字`

- 当前时间格式：`MMDDHH`（月+日+小时）
- 输出统一为 **6位纯数字密码**

## 🚀 使用方法
1. 下载项目中 `index.html` 文件
2. 使用任意浏览器直接双击打开
3. 方式一：手动输入8位SOP日期 `YYYYMMDD`
4. 方式二：点击日历图标 📅 快速选择出厂日期
5. 点击 **计算密码** 即可立即生成
6. 点击密码旁「复制」按钮一键粘贴使用
7. 网页保持打开，每小时自动刷新最新密码
